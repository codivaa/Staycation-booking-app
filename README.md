# Staycation Booking App

A modern multi-step booking web application built using React and Vite.
This project simulates a real-world vacation booking experience with a smooth and structured user flow from property selection to payment confirmation.

---

## Features

* Multi-step booking process (Details → Booking → Payment → Confirmation)
* Structured navigation flow across pages
* Interactive user interface with intuitive navigation
* Responsive and clean UI design
* Designed user journey to simulate a real booking experience

---

## Tech Stack

* React (Vite)
* JavaScript (ES6+)
* CSS
